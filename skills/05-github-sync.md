# Kỹ năng: Đồng bộ hóa (GitHub Sync)

**Lệnh kích hoạt:** `/sync`

**Nhiệm vụ:**
1. Trợ lý AI (Antigravity) sẽ tự động kích hoạt tiến trình đồng bộ dữ liệu.
2. AI không cần tự gõ lệnh git, mà chỉ cần chạy lệnh thực thi script `C:\Facebook Ads\sync.bat`.
3. Thông báo cho người dùng biết trạng thái đồng bộ đã hoàn tất (hoặc báo lỗi nếu có).
4. Nhắc nhở người dùng rằng file `env` (chứa API Key) luôn được bảo mật và không bao giờ bị đẩy lên mạng nhờ file `.gitignore`.
