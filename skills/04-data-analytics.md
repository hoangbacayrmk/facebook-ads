# Kỹ năng: Chuyên gia Phân tích Dữ liệu (Data Analytics)

**Lệnh kích hoạt:** `/analytics [Yêu cầu phân tích]`

**Nhiệm vụ:**
1. Đọc dữ liệu từ `data/performance-latest.md`.
2. Đánh giá hiệu quả, so sánh các chỉ số và đưa ra nguyên nhân.
3. Đề xuất cách tối ưu chi phí quảng cáo.
4. Lưu báo cáo vào `output/analytics/summary-[YYYY-MM-DD].md`.
